LeoCaptcha

Simple, Easy yet Powerful CAPTCHA

Made by Malay Bhavsar

How to Implement

1. Download the file and add it to your project

2. Link the file in the head section of your page

<script src="./captcha.js"></script>

3. Add the following HTML snippets at the place for CAPTCHA

<canvas id="myCanvas"></canvas>
<input
    type="text"
    name="CaptchaText"
    id="myInput"
    placeholder="Enter Captcha"
/>
<input
    type="button"
    onclick="captcha.displayCaptcha()"
    value="Reload"
    id="primary"
/>
<input
    type="button"
    onclick="captcha.verifyCaptcha(captchaSuccess, captchaFailure)"
    value="Submit"
    id="success"
/>

4. Add the following JavaScript Snippets.

let captcha = new LeoCaptcha("myCanvas","myInput","AlphaNumeric", 5);
captcha.displayCaptcha();

function captchaSuccess() {
    // This function will be executed on Success
}
function captchaFailure() {
    // This function will be executed on Failure
}

5. Apply CSS according to your Needs

Code Description Example
cap.setDimension(width, height) Pass the desired height and width of CAPTCHA Canvas cap.setDimension() // Default
cap.setDimension(300, 60)
cap.setColor(enableColor, colorArray) If you have a specific color choice ten pass in the form of array or if you dont want them to be colorful then pass false which defaults to Black color Text cap.setColor() // Default
cap.setColor(false) // Disable
cap.setColor(true, ["#f00"]) // Single
cap.setColor(true, ["#f00", "#0f0", "#00f"]) // Multiple
cap.setRotate(enableRotate, rotateArray) Pass false if you want you text be in one line and not jumbled up and done.You can also pass the array for the rotation value from -0.07 to 0.07 cap.setRotate() // Default
cap.setRotate(false) // Disable
cap.setRotate(true, [-0.07, 0, 0.07]) // Custom
cap.setBgColor(enableBgColor, bgColorArray) If you want your background to be of a specific color then pass the color array of your choise. But if you choose to pass false then the it defaults to white background. cap.setBgColor() // Default
cap.setBgColor(false) // Disable
cap.setBgColor(true, ['#fff']) // Single
cap.setBgColor(true, ['#fff','#ddd','#bbb']) // Multiple
